## 🚀 Quick Start

### Option 1: Source Code Deployment (Recommended)

#### Prerequisites

| Tool | Version | Check |
|------|---------|-------|
| **Node.js** | 18+ | `node -v` |
| **Python** | ≥3.11, ≤3.12 | `python --version` |
| **uv** | Latest | `uv --version` |

#### 1. Configure Environment

```bash
cp .env.example .env
# Edit .env with your API keys
```

**Required Environment Variables:**

```env
# LLM API (supports any OpenAI SDK-compatible API)
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus

# Zep Cloud (Knowledge Graph)
ZEP_API_KEY=your_zep_api_key
```

#### 2. Install Dependencies

```bash
npm run setup:all
```

#### 3. Start Services

```bash
npm run dev
```

**Service URLs:**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:5001`

### Option 2: Docker

```bash
docker compose up -d
```

## 🔮 Purple Star Astrology API

### Birth Chart

```bash
# Generate chart
POST /api/divination/chart/generate
Content-Type: application/json

{
  "birth_info": {
    "name": "John",
    "gender": "male",
    "birth_date": "1990-08-20",
    "birth_time": "12:00",
    "birth_location": "Beijing"
  }
}

# Get chart
GET /api/divination/chart/<chart_id>
```

### Analysis

```bash
POST /api/divination/agents/analyze
{
  "chart_id": "uuid",
  "analysis_types": ["stars", "palaces", "transforms", "patterns", "timing"]
}
```

### Report Generation

```bash
POST /api/divination/report/generate
{
  "chart": {...},
  "user_name": "John",
  "year": 2026,
  "report_type": "professional_plain"
}
```
